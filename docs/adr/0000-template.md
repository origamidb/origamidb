## Context

State the problem in one or two paragraphs. Cite project sources verbatim where they bear on the decision (e.g., a quote from `ARCHITECTURE.md`, a prior ADR, or an accepted RFC). If you reference a claim that is not grounded in a stated project source, mark it `*inferred:*` and identify what the inference rests on.

## Considered options

## Decision

## Consequences
