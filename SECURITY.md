# Security Policy

## Supported Versions

OrigamiDB is pre-1.0 and under active development. 
There are no production-ready releases at this time. 
Security fixes will be applied to the latest main branch only.

## Reporting a Vulnerability

Please do not report security vulnerabilities through public GitHub issues.

Instead, use the GitHub Security Advisory 
["Report a Vulnerability"](https://github.com/origamidb/origamidb/security/advisories/new) 
tab to report privately.

You can expect an acknowledgement within 5 business days. 
We will keep you informed as we work toward a fix and will 
credit your contribution when the issue is disclosed publicly.

If you do not receive a response within 5 business days, 
you can reach us directly at security@asaddu.com.
